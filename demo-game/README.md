# 横スクロールアクションゲーム

スーパーマリオ風の横スクロールアクションゲームです。

## 遊び方

1. `index.html` をブラウザで開く
2. 矢印キーで左右に移動
3. スペースキーでジャンプ
4. 敵と障害物を避けてスコアを稼ぐ

## 操作方法

- **← →** : 左右移動
- **スペースキー** : ジャンプ
- **R キー** : リスタート

## ゲームオーバー条件

- 敵にぶつかる
- 障害物にぶつかる
- 画面下に落ちる

## 技術スタック

- HTML5 Canvas
- Vanilla JavaScript
- CSS3

## ファイル構成

- `index.html` - ゲームのHTMLページ
- `game.js` - ゲームロジック
- `README.md` - このファイル
